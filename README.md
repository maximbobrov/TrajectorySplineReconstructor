To run the project, place the folder with the ".dat" files in the assembly folder.
You can change the view using the mouse and the "w, a, s, d, q, e" keys.
Pressing the "c" key performs one iteration of functional minimization.
Pressing the "v" key performs full functional minimization to the accuracy specified in the SplineInterpolator class constructor.
Pressing the "1, 2, 3, 4, 5" keys enables/disables the drawing of points, polylines, splines, velocity vectors, and acceleration vectors, respectively.
Pressing the "[, ]" keys increases/decreases the lengths of the velocity and acceleration vectors.
